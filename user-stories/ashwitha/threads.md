# Threads (Member Communication)

## As
An EBD Staff member

## I want to
View and manage communication threads with members

## So that
I can respond to member queries and track ongoing conversations

## Acceptance Criteria
- I can see a list of threads after logging in and navigating to Threads
- Each thread shows a subject or title
- Clicking a thread opens the full conversation view
- Thread list loads without errors

## Negative Cases
- Access thread list without authentication → 401 Unauthorised
