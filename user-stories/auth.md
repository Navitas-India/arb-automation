# Authentication

## As
An EBD Staff member

## I want to
Log in to the ARBenefits admin portal with my credentials

## So that
I can access member data, enrollment records, and admin features

## Acceptance Criteria
- I can enter my email and password on the login page
- Clicking Sign In with valid credentials redirects me to the dashboard
- No error message appears after successful login

## Negative Cases
- Wrong password → error message shown, stay on login page
- Empty fields → validation error shown
- Non-existent email → error message shown
